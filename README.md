# CityRide-Scooter-Dashboard
# Scooter Revenue & Pricing Dashboard (Power BI)

## Business Question
Can the company increase prices further without reducing demand?

## Overview
This project analyses revenue, profit, and demand trends for a scooter-sharing business using Power BI.

## Tools Used
- Power BI
- SQL
- Excel

## Key Insights
- A 25% price increase led to a 64% increase in demand
- Demand showed low sensitivity to price changes
- Recommended a 10–15% price increase as a safer strategy

## Dashboard Features
- KPI cards (Revenue, Profit, Riders)
- Trend analysis over time
- Interactive slicers for filtering
